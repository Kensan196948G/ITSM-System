## 詳細確認項目

### アニメーション確認

#### サイドバーのスライド
1. ハンバーガーボタンをクリック
2. サイドバーが**0.3秒**かけて左から右にスライド
3. 動きが**スムーズ**であることを確認
4. カクつきやちらつきがないことを確認

#### オーバーレイのフェード
1. サイドバーを開く
2. オーバーレイが**0.3秒**かけてフェードイン
3. 透明度が0から1に滑らかに変化
4. サイドバーを閉じる時も滑らかにフェードアウト

---

### レスポンシブ挙動確認

#### ブラウザリサイズ時の動作

**デスクトップ → タブレット**
1. デスクトップサイズから開始（横幅800px以上）
2. ブラウザ幅を徐々に狭くする
3. **768px**を境に：
   - [ ] ハンバーガーボタンが表示される
   - [ ] サイドバーが非表示になる
   - [ ] レイアウトが切り替わる

**タブレット → デスクトップ**
1. サイドバーを**開いた状態**でブラウザ幅を広げる
2. **768px**を超えると：
   - [ ] ハンバーガーボタンが非表示になる
   - [ ] サイドバーが常時表示に戻る
   - [ ] オーバーレイが消える
   - [ ] レイアウトがデスクトップ用に戻る

**タブレット → モバイル**
1. タブレットサイズから開始（横幅600px程度）
2. ブラウザ幅を徐々に狭くする
3. **480px**を境に：
   - [ ] ダッシュボードグリッドが1列になる
   - [ ] フォントサイズが縮小される
   - [ ] ボタンサイズが拡大される

## トラブルシューティング

### 問題: ハンバーガーボタンが表示されない

**確認事項**
1. ブラウザ幅が768px以下であることを確認
2. DevToolsでデバイスモードが有効か確認
3. ページを再読み込み（`Ctrl + F5`または`Cmd + Shift + R`）

### 問題: サイドバーが開かない

**確認事項**
1. ブラウザのコンソールでJavaScriptエラーがないか確認
2. `initMobileNavigation()` 関数が実行されているか確認
3. `#sidebar-toggle` 要素が存在するか確認

**コンソールでの確認**
```javascript
// ブラウザのコンソールで実行
document.getElementById('sidebar-toggle')  // nullでないことを確認
document.querySelector('.sidebar')         // 要素が存在することを確認
```

### 問題: アニメーションがカクつく

**確認事項**
1. ブラウザのハードウェアアクセラレーションが有効か確認
2. 他のタブやアプリを閉じてCPU負荷を下げる
3. 古いブラウザの場合は最新版にアップデート
