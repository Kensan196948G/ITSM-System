# NIST_CSF2.0準拠要件（NIST-CSF2.0-Compliance-Requirements）

## 1. GOVERN (統治)
- サイバーセキュリティ方針の確立
- リスク管理戦略の定義

## 2. IDENTIFY (識別)
- 資産管理 (ID.AM)
- 脆弱性識別 (ID.RA)

## 3. PROTECT (保護)
- ID管理とアクセス制御 (PR.AA)
- データ保護 (PR.DS)

## 4. DETECT (検知)
- 継続的なセキュリティ監視 (DE.CM)
- 異常の発見 (DE.AE)

## 5. RESPOND (対応)
- 対応計画の実行 (RS.RP)
- インシデントの分析 (RS.AN)

## 6. RECOVER (復旧)
- 復旧計画の実行 (RC.RP)
- 公報および改善 (RC.CO)
