# 機能要件（Functional-Requirements）

## 1. ダッシュボード機能
- サービスレベルのリアルタイム監視
- セキュリティスコア（NIST CSF 2.0 機能別）の表示
- アラートおよび通知機能

## 2. インシデント管理 (ISO 20000 / CSF RESPOND)
- チケット発行・追跡
- 優先度自動判定
- セキュリティインシデントと標準インシデントの区分け

## 3. 構成管理 (CMDB / CSF IDENTIFY)
- IT資産（ハードウェア・ソフトウェア）のリスト管理
- 重要資産のタグ付け
- 依存関係の可視化

## 4. 変更管理 (ISO 20000 / CSF PROTECT)
- 変更要求 (RFC) の承認ワークフロー
- リスク評価の統合
