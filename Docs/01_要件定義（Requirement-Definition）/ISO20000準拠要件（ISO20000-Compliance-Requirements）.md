# ISO20000準拠要件（ISO20000-Compliance-Requirements）

## 1. サービス管理システム (SMS) 要求事項
- 方針、役割、責任の定義
- リソースの提供

## 2. サービスの運用プロセス
- **8.2.6 構成管理**: 構成品目 (CI) の定義と管理
- **8.5.1 変更管理**: 変更の影響評価と承認
- **8.6.1 インシデント管理**: インシデントの記録、分類、解決
- **8.6.2 問題管理**: 根本原因の特定と予防

## 3. 監視と測定
- SLA (Service Level Agreement) の遵守率測定
- 顧客満足度の測定（将来拡張）
